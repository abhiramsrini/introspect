Updating Rx Phase calibration and validation for SV5C-CPRX (31-07-2025)
----------------------------
Description:
- The Rx phase calibration previously tried to match the sync offset of bertScans across all the wires, before measuring the phase delays, but this was prone to failure.
- Now the algorithm will sweep the rx hsThreshold, and check that the eyewidth of the bertScan is within 85% of the UI before measuring the phase delays. This ensures that the bertScan edges are representative of the eye-crossing, without necessitating that the sync offsets are exactly the same.
- The validation step will validate the phase delays at 4.5Gsps in part 1, and then measure the combined eyes in part 2 as a sanity check that the wires are properly aligned.

IntrospectIESP version 23.1.0
----------------------------
Hardware
    - SV5C-CPRX unit to calibrate
    - SV5C-CPTX
    - MXP cable
    - Connect the external clock of SV5C-CPRX to SV5C-CPTX
----------------------------
Description
The calibration is divided into two parts:
    - Find the threshold offsets: 
      Run eye scan for each wire at 4500 Mbps. 
	  The offset is obtained by taking the average offset voltage from the position of the eye center at two states (even/odd).
    - Find the phase offset: 
      It is done by doing a bert scan for each wire. The data rate used is 4000 Msps. 

The phase validation is divided into three parts for debugging purposes:
    - SV5C_CPRX_phase_Val_Analyzer_part1: validate the phase offsets at 4000 Msps
    - SV5C_CPRX_phase_Val_Analyzer_part2: run individual eye scans for each wire at 4500 Msps
    - SV5C_CPRX_phase_Val_Analyzer_part3: run combined eye scans at 4500 Msps
----------------------------
Steps of the calibration
    1- Run Scripts/sv5RxGenerateFlash.bat
    2- Start the threshold calibration (folder: rxThresholdCal)
    3- Run Scripts/sv5RxGenerateFlash.bat
    4- Start the threshold validation (folder: rxThresholdVal)
    5- Start the phase calibration (folder: rxPhaseCal)
    6- Run Scripts/sv5RxGenerateFlash.bat
    7- Start the phase validation (folder: rxPhaseVal):
       7.1- Start SV5C_CPRX_phase_Val_Analyzer_part1
       7.2- Start SV5C_CPRX_phase_Val_Analyzer_part2
       7.3- Start SV5C_CPRX_phase_Val_Analyzer_part3
----------------------------
Notes:
* Changed condition for the eyeHeight from 370 mV to 350 mV during threshold calibration/validation. 
  This is to prevent the threshold offset to be based on an incomplete eye.
- A popup dialog where the user needs to enter the serial number of the RX unit. 
  Expected format is 89-xxxx. If the provided serial number is not as expected, the calibration won't start.
- The result folder will be named as the serial number provided. (In case you don't see the folder in the GUI, try to refresh Results)
- Before programming the calibration .jam file using Scripts/sv5RxGenerateFlash.bat, please turn off the generator unit.
